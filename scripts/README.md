# Вспомогательные скрипты

Эта папка содержит вспомогательные скрипты для работы с проектом.

## Скрипты

### Управление статическими файлами

- **`download_static_assets.py`** - Автоматическая загрузка CSS/JS файлов с CDN
  ```bash
  python3 scripts/download_static_assets.py
  ```

- **`extract_static_files.py`** - Установка статических файлов из папки static_downloads
  ```bash
  python3 scripts/extract_static_files.py
  ```

- **`download_links.html`** - HTML страница со ссылками для ручного скачивания файлов

### Тестирование

- **`test_mode.py`** - Модуль для тестового режима работы без реальной БД
- **`test.py`** - Вспомогательные тестовые скрипты

## Использование

Все скрипты запускаются из корневой директории проекта:

```bash
python3 scripts/script_name.py
```

Скрипты автоматически определяют корневую директорию проекта, поэтому их можно запускать из любого места.
