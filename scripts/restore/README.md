# Скрипты восстановления данных

Эта папка содержит скрипты для восстановления удаленных жестких дисков через API.

## Файлы

- **restore_disks_simple.py** - Скрипт восстановления дисков (использует встроенный urllib)
- **restore_disks.py** - Скрипт восстановления дисков (использует requests, требует установки)
- **restore_disks.json** - JSON файл с данными для восстановления
- **restore_disks.sh** - Bash скрипт для восстановления (требует curl)

## Использование

### Python (рекомендуется)
```bash
python3 restore_disks_simple.py
```

### Bash (требует curl)
```bash
chmod +x restore_disks.sh
./restore_disks.sh
```

## Формат данных

Файл `restore_disks.json` содержит данные о дисках в формате API `/api/hdd_collect`:
- `hostname` - имя компьютера
- `disks` - массив объектов с данными о дисках

## Примечание

Скрипты автоматически восстанавливают деактивированные диски при обновлении через API.
