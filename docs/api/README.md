# API Документация

Эта папка содержит документацию по API endpoints проекта.

## Файлы

- **API_HDD_ENDPOINTS.md** - Документация по API v1 для работы с жесткими дисками (создание, обновление, автоматический сбор данных)
- **API_HDD_ENDPOINTS_V2.md** - Документация по API v2 для работы с компьютерами и жесткими дисками (расширенная версия с информацией о машинах, ОС, железе)

## Содержание

### API для жестких дисков

**API v1** (`/api/hdd_collect`):
- Создание жестких дисков через веб-форму
- Редактирование жестких дисков через веб-форму
- Автоматический сбор данных через JSON API
- Автоматическое определение производителей
- Конвертация статусов здоровья
- Создание записей в истории состояний

**API v2** (`/api/hdd_collect/v2`) - Рекомендуется:
- Все возможности API v1
- Информация о компьютерах/машинах (hostname, IP, MAC)
- Информация об операционной системе
- Информация об аппаратном обеспечении (процессор, память, материнская плата)
- Сетевая информация (домен, роль компьютера)
- История изменений конфигурации машин
- Связь дисков с машинами

## Использование

Все API endpoints описаны с примерами запросов (cURL и Python) и форматами ответов.
