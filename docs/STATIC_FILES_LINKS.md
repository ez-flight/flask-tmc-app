# Ссылки для скачивания статических файлов

Скачайте следующие файлы в Windows и поместите их в папку `static_downloads/` в корне проекта:

## 1. Bootstrap 5.3.0

**CSS:**
- Ссылка: https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css
- Сохранить как: `bootstrap.min.css`

**JavaScript:**
- Ссылка: https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js
- Сохранить как: `bootstrap.bundle.min.js`

## 2. Bootstrap Icons 1.11.0

**CSS:**
- Ссылка: https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.0/font/bootstrap-icons.css
- Сохранить как: `bootstrap-icons.css`

**Шрифты:**
- WOFF: https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.0/font/fonts/bootstrap-icons.woff
- Сохранить как: `bootstrap-icons.woff`

- WOFF2: https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.0/font/fonts/bootstrap-icons.woff2
- Сохранить как: `bootstrap-icons.woff2`

## 3. Chart.js 4.4.0

**JavaScript:**
- Ссылка: https://cdn.jsdelivr.net/npm/chart.js@4.4.0/dist/chart.umd.min.js
- Сохранить как: `chart.umd.min.js`

## 4. TinyMCE 6 (Self-hosted, GPL версия)

**JavaScript:**
- Ссылка: https://download.tiny.cloud/tinymce/community/tinymce_6.8.3/tinymce.min.js
- Сохранить как: `tinymce.min.js`

**Примечание:** Используется self-hosted GPL версия, которая не требует API ключ.

---

## Инструкция по скачиванию:

1. Создайте папку `static_downloads` в корне проекта
2. Скачайте все файлы по ссылкам выше
3. Сохраните их с указанными именами в папку `static_downloads/`
4. Запустите скрипт `extract_static_files.py` для автоматической установки

## Альтернативный способ - скачать все сразу:

Вы можете использовать браузер с расширением для массового скачивания или создать HTML файл со всеми ссылками.
